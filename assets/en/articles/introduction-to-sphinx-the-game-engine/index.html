Introduction to Sphinx

作者 / Sphinx

High efficiency, One-time running, No app authentification from app store, Seamless updating on server, these are some great side of HTML5 games, yet we can not both have two side advantages. So the performance of HTML5 game is always a big problem to game developers. Normally a cool effect game can only run less than 30 frame per second on mobilde device, not even thinking about compare to a native app, in this situation, the game player could very fretful on your game. What if a web game engine can make the performance of web game like that of a native game?
Well, upon the development in one year, we are now announce the Sphinx Game Engine, a game engine particularly optimized for HTML5 games, it is possible to have the performance and the great side of HTML5 games. Sphinx makes it happen!
Sphinx use the original Presto (Opera) layout engine, its small and fast, and based on it, we redesign the rendering module and use the asyncronized rederening mode of OpenGL improve the graphics renderation, due to makes to game run on device high efficiency and performance.
Otherwise, we also have some graet remote debug tools, it'll make developers run their HTML5 code on device remotely. The debugger will not only contain the traditional break-point or single-step debugging kind of feature, it also have a feature called "Profile" - to make developers read the detail of how code is runing on which process.

Sphinx 完全基于 HTML5 标准，支持 HTML5 各种原有的特性，比如 DOM、Web Socket、Device API 等，这些让 HTML5 开发者无需修改自己的代码仅仅是使用 Sphinx 就可以轻松的获得很高的性能。现在流行的游戏引擎框架比如 Cocos2d-html5、CreateJS 都可以很好的跑在 Sphinx 上。
现在 HTML5 开发者可以通过我们发布的 Sphinx 浏览器，直接在 Sphinx 里面输入游戏的 URL，便可以看到自己的游戏在 Sphinx 上跑的效果。
同时为了让 HTML5 代码能够摇身一变，变成像 native app 的 apk，我们还发布了打包服务，开发者只要如下几步就可以轻松将现有的 HTML5 游戏转变成 apk。为了保护 HTML5 开发者的版权，打包的时候还会对游戏加密。


Sphinx 支持灵活的商务合作模式，并可以基于欧朋的渠道向 CP 提供系列商业推广服务，开发者可通过 Sphinx 开发者平台管理系统注册 CP 账号，获取一般性打包服务，也可通过 sphinx@oupeng.com 邮箱与欧朋联系合作事宜，获得专属技术＆商务服务。
