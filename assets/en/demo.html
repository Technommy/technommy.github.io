Demo 1: 不江湖

（双击全屏）
一款基于 CreateJS 框架的大型多人在线角色扮演类游戏。游戏中人物行走、对话、战斗中的招式全部由序列帧动画组成。贴图资源在百兆量级，支持本地和远程两种加载方式，支持远程增量透明更新。游戏场景使用 HTML5 Canvas2d API 绘制，对话框等则采用传统的 HTML + CSS 技术，网络通信基于 WebSocket 实现。游戏的整体像素填充率平均在 1.5～2.5 倍屏幕尺寸。在主流中高端机型（小米 2、三星 S3／S4）上可以维持在 40〜50 fps 之间。

Demo 2: Moon Warriors

Moon Warriors 是知名的 2D 游戏引擎 Cocos2d 的官方示例。Sphinx 为 Cocos2d-html5 引擎做了专门的适配与调优，所有基于 Cocos2d-html5 开发的游戏在 Sphinx 中可以获得更佳的性能和体验。同时，我们也对 Cocos2d 的源代码进行了部分优化，以更好的适配主流的 JavaScript 引擎。

Demo 3: Monsters

Monsters 是基于 HTML5 Canvas2d 的位图测试，代表了典型的 2D 游戏场景。场景中包含了 100 个精灵（monster），精灵的位置在每一帧中都会刷新，这个游戏的帧率能够很好的说明设备在一个场景能够同时处理图元的数量。如果一个游戏的复杂度和这个示例相当（主场景中包含 100 个左右的图元），理论上可以获得和这个示例相当的性能。

Demo 4: Cut the Rope

Cut the Rope（割绳子）是一款老少咸宜的休闲游戏，兼容众多平台。其 HTML5 移植版本充分保留了原作的交互性和娱乐性，同时也证明了 JavaScript 有能力进行复杂运算（基于物理的运动模型），并保证一定的性能。本示例为移动设备进行了专门的适配，原版请参考官方网页。建议使用分辨率 1280x800 的设备运行，以获得最佳体验。
