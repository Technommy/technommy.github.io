Sphinx: A Born HTML5 Game Engine

Specification Developer tool Download emulator Debug tool (Opera 12)

The lack of smooth experience in a mobile HTML5 game is always a big problem for game developers, as with normal browsers a HTML5 game runs in less than only 30 frames per second. Well, what if a web game engine can make the performance of web game like that of a native game? Developers will no longer need to worry about performance issue and all that sleepless nights. That's why we build Sphinx, a game engine particularly optimized for HTML5 games.
