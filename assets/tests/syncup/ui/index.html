UI Redesign for SyncUp (Overview)

#

Mindmap & Sketch

页面／功能树形图
上图为 SyncUp 的基础功能树形图，带有 (?) 的分项待一起讨论。目前的 UI 设计只包含了之前存在的主页（homepage）和功能页面（video page），在有后续功能时会在当前设计基础上加入。
Join A Session / Create A Session 功能草图
之前的 Video Page 将页面划为三部分，其中右侧的讨论区（chat area）是点击右上角的「人物」从右向左划出。重新设计后的方案有三种，如下图：
上面线框图每种分法都包含了一个顶部栏（topbar）
第一种为原先的界面线框图
第二种将页面划为三部分，包含了一个视频信息和社交服务分享区块
第三种将页面划为两部分，与第一种不同是不用进行点击／滑动打开右边的「Main」区，固定在页面
考虑到内容划分，绿色区块的排版没有较好的方案，目前使用的是第三种。
注：以上划分在不同显示器下均保持百分比固定的显示效果。
#

UI Base

颜色: #2c3e50 (Midnight Blue) / #27ae60 (Nephritis)，待讨论。
标识: 目前是拿 Source Sans Pro 来作 SyncUp 的标识，左边 Icon 是从 Ionicon 库中取的。
字体: 页面调用 Source Sans Pro 作为默认字体（一些小部件字体可能会为 Arial 以保持辨识度），另外拿 Ionicon 和 FontAwesome 作图标库。
